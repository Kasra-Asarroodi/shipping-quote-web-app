# Future Improvements

Although the application is fully functional, there are many improvements that could be added in future versions.

## Possible Improvements

- User account registration.
- Password reset functionality.
- Email confirmation after submitting an enquiry.
- Online payment integration.
- Administrator analytics dashboard.
- Automatic quotation PDF generation.
- Search and filtering within enquiries.
- Export enquiries to Excel.
- Mobile responsive improvements.
- Improved form validation.
- PostgreSQL migration.
- Automated testing.

## Long-Term Goal

The long-term goal is to continue improving this application as I progress through university.

Rather than treating it as a completed assignment, I see this as a real software project that can continue to evolve while I learn new technologies.

As I gain experience with authentication, APIs, cloud services and larger databases, I plan to integrate those skills into future versions of this application.
